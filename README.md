# MeetLove - 情緒分析配對系統

## 專案概述

**產品名稱：** MeetLove  
**核心目標：** 透過 Web 1對1視訊對話，蒐集並分析雙方的對話內容與面部表情，判斷雙方喜好程度  
**輸出狀態：** High / Normal / Low 三種情緒狀態

---

## 研究主旨

**論文標題：** 個人情緒構成分析及現況情緒反應決策輔助判斷系統

### 研究動機
- 現代人對於情感溝通需要客觀輔助工具
- 透過 AI 分析可提供個人情緒狀態的自我檢測
- 幫助使用者了解自己在互動中的情緒變化

### 研究目標
1. 建構情緒構成分析模型
2. 開發即時情緒反應判斷系統
3. 驗證系統在實際對話情境中的準確性

---

## 功能模組

| 模組 | 功能說明 |
|------|----------|
| **視訊模組** | WebRTC 1對1即時視訊通話 |
| **表情分析** | 即時臉部情緒辨識（開心、悲傷、生氣、驚訝等） |
| **語音分析** | 語調、語速、情緒音調分析 |
| **文字分析** | 對話內容情感分析 |
| **整合判斷** | 綜合三種輸入源的喜好判斷 |
| **狀態輸出** | High / Normal / Low 三種狀態 |

---

## 技術架構（初稿）

### 前端
- React + TypeScript
- WebRTC (Simple-Peer 或 PeerJS)
- TensorFlow.js (臉部表情)
- Web Speech API (語音辨識)

### 後端
- Node.js + Express
- Socket.io (即時通訊)
- Python (可選，用於進階 ML 模型)

### ML/AI
- 表情：Face-api.js 或 MediaPipe
- 語音：Web Audio API + 情緒分析模型
- 文字：Sentiment analysis (TensorFlow.js 或 NLP.js)
