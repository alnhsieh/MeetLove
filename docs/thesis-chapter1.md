# 第一章 緒論

## 1.1 研究背景與動機

隨著科技的快速發展與網際網路的普及，人際互動的模式已經發生了翻天覆地的變化。從傳統的面對面交流，逐漸轉變為以文字、語音、視訊為主的數位溝通方式。特別是近年來，視訊通話軟體如 Zoom、Google Meet、Microsoft Teams 等已經成為人們日常生活與工作中不可或缺的工具。然而，現有的視訊通話系統主要著重於提供穩定的通訊品質，卻忽略了在溝通過程中對於雙方情緒狀態的理解與分析。

在情感交流的層面上，人類對於對方的情緒反應往往是非常敏感的。一個微笑、一個眼神或是語調的變化，都可能傳達出豐富的情感資訊。然而，在遠距溝通的環境中，由於缺乏面對面的互動，我們往往難以準確地捕捉和解讀這些細微的情緒信號。這種資訊的不對稱往往會導致溝通效果的下降，甚至造成誤解。

近年來，人工智慧技術的蓬勃發展為解決上述問題提供了新的可能性。特別是在電腦視覺、自然語言處理以及語音處理等領域，各種深度學習模型已經達到了與人類相當甚至超越人類的水準。情緒分析（Emotion Analysis）作為人機互動領域的重要研究方向，已經在多個應用場景中展現出其價值，包括客服系統、健康醫療、教育輔助以及社交網路分析等。

此外，現代人對於自我身心健康的關注度日益提升。越來越多的人開始注重自己的情緒管理與心理健康。然而，傳統的情緒評估方式往往需要專業的心理師介入，不僅成本較高，且難以做到即時、私密且持續的自我監測。因此，開發一套能夠在日常互動中自動分析個人情緒狀態的系統，對於促進自我覺察與心理健康維護具有重要的意義。

本研究的動機正是基於上述背景：我們希望能夠開發一套結合多模態人工智慧技術的系統，透過分析使用者在視訊對話中的表情、語音以及文字內容，即時且準確地判斷使用者的情緒狀態，並提供個人情緒構成的分析報告。這種技術不僅可以應用於情感配對、社交互動等場景，還可以作為個人情緒自我監測的工具，幫助使用者更好地了解自己的情緒變化模式。

## 1.2 研究目的

本研究旨在開發一個名為「MeetLove」的個人情緒構成分析及現況情緒反應決策輔助判斷系統。具體而言，本研究的目的可分為以下幾個面向：

### 1.2.1 建構多模態情緒分析框架

本研究的首要目標是建立一個整合表情分析、語音分析以及文字分析的多模態情緒分析框架。我們相信，單一情緒來源往往難以完整地表達一個人的真實情緒狀態，因此透過融合多種輸入源，可以獲得更加準確和全面的情緒評估結果。

在表情分析方面，我們採用 MediaPipe Face Mesh 技術進行動態臉部特徵點偵測，並基於這些特徵點進一步分析使用者的面部表情變化。在語音分析方面，我們則透過分析語調、語速以及聲音能量等聲學特徵，來推斷使用者的情緒狀態。在文字分析方面，我們利用自然語言處理技術，對對話內容進行情情感分析，從文字層面理解使用者的情緒傾向。

### 1.2.2 開發即時情緒反應判斷系統

除了靜態的情緒分析之外，本研究還致力於開發一套能夠即時反應使用者情緒變化的系統。這意味著我們的系統必須能夠在對話進行的同時，持續地分析並輸出情緒狀態的更新。為此，我們必須優化演算法效率，確保系統能夠在可接受的延遲範圍內完成分析。

此外，我們還設計了一套基於加權融合的決策演算法，將來自表情、語音以及文字三種不同來源的情緒分數進行整合，最終輸出 High、Normal、Low 三種級別的情緒狀態。這種分類方式不僅直覺易懂，也能夠讓使用者快速地理解當前的情緒狀態。

### 1.2.3 實現個人情緒構成分析

本研究的另一個重要目的是實現個人情緒構成分析功能。透過收集並分析使用者在多次對話中的情緒數據，我們希望能夠建立一套個人情緒檔案，記錄使用者的情緒波動模式、情緒觸發因素以及情緒恢復時間等資訊。這些資訊對於帮助使用者了解自己的情緒構成，以及識別潛在的情緒問題具有重要的價值。

### 1.2.4 開發 MVP 原型系統

最後，本研究也將實際開發一個名為 MeetLove 的 MVP（Minimum Viable Product，最小可行性產品）原型系統。這個系統將會是一個 Web 應用程式，支援一對一的視訊對話功能，並在其中嵌入我們開發的情緒分析模組。使用者可以透過瀏覽器直接使用這套系統，無需安裝額外的軟體。我們希望透過這個原型系統，驗證我們所提出的情緒分析方法的可行性，並收集實際使用者的回饋以進一步優化我們的技術。

## 1.3 論文貢獻

基於上述研究目的，本論文預計將產生以下幾項主要貢獻：

### 1.3.1 提出整合表情、語音、文字的情緒分析框架

本研究提出了一個創新的多模態情緒分析框架，該框架能夠同時處理表情、語音以及文字三種不同類型的輸入，並透過加權融合的方式輸出最終的情緒判斷結果。與現有的單一模態情緒分析系統相比，我們的框架能夠提供更加準確和可靠的情緒評估。我們也會詳細說明各模組的技術實現細節，包括臉部偵測、特徵提取、情緒分類以及融合決策等各個環節。

### 1.3.2 設計並實現 High/Normal/Low 三級喜好判斷機制

本研究提出了一種直覺且實用的情緒分類方式，將使用者的情緒狀態分為 High、Normal、Low 三個等級。這種分類方式不同於傳統的細緻情緒分類（如快樂、悲傷、憤怒等），雖然犧牲了情緒類型的多樣性，但卻能夠提供更加明確的判斷結果，方便一般使用者理解和使用。同時，我們也會說明這種分類方式的設計理念以及閾值的選擇依據。

### 1.3.3 開發 Web-based 即時視訊配對原型系統

本研究實際開發了一套基於 Web 技術的即時視訊配對原型系統 MeetLove。這個系統使用現代化的技術堆疊，包括 React、FastAPI、WebRTC 以及 TensorFlow.js 等，能夠在瀏覽器中實現流暢的視訊通話體驗，並同時進行即時的情緒分析。我們會在論文中詳細說明系統的架構設計、技術選型以及實現過程，為後續的開發者提供完整的參考。

### 1.3.4 驗證多模態情緒分析在實際應用場景中的有效性

最後，我們會透過實際的使用者實驗，驗證我們所提出的多模態情緒分析框架在真實應用場景中的有效性。我們將收集實驗參與者在使用 MeetLove 系統時的情緒數據，並從準確率、延遲時間以及使用者滿意度等多個維度進行評估。這些實驗結果將為我們的方法提供實證支持，同時也能夠發現系統的不足之處，為未來的優化方向提供指引。

## 1.4 論文架構

本論文共分為五章，各章節的內容安排如下：

第一章為緒論，說明本研究之背景與動機、研究目的、預期貢獻以及論文架構。

第二章為文獻探討，回顧與本研究相關的國內外文獻，包括情緒分析理論、臉部表情辨識技術、語音情緒分析、文字情感分析以及多模態情緒識別等相關領域的最新進展。同時也會分析現有的類似產品，並說明本研究與現有研究之差異。

第三章為系統設計與方法，詳細介紹本研究所開發之 MeetLove 系統的整體架構、各功能模組的設計原理以及所使用的技術方法。

第四章為實驗設計與結果，說明本研究之實驗設計、資料收集方式、模型訓練過程以及系統效能評估的結果。

第五章為結論與未來工作，總結本論文的研究成果，並提出研究限制以及未來可能的研究方向。
